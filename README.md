### STEP 3 : VMGit
-  DVCS 분산 저장소 방식에 대해 학습하고, 기본 동작을 구현
- init : 저장소를 init 하고 checkout 으로 이동하는 과정을 구현
- add, commit : 파일을 생성하고, 추가하고, 커밋하는 흐름을 작성
- push : 로컬에 커밋한 내용을 remote에 반영하는 흐름을 구현
- clone : remote에 있는 저장소를 local에 가져오는 방식을 구현
